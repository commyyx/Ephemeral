## QUICK START

```bash
# 1. Estrai archivio
unzip securechat_modular.zip    # Windows
tar -xzf securechat_modular.tar.gz   # Linux/Mac

# 2. Entra nella directory
cd securechat_modular

# 3. Installa dipendenze
npm install

# 4. Test (opzionale)
node test.js

# 5. Avvia server
npm start

# 6. Apri browser
http://localhost:3000
```
